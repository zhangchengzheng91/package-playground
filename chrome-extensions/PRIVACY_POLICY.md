# CSS Style Injector - Privacy Policy

**Last Updated: December 2024**

## Information We Collect

CSS Style Injector does not collect, store, or transmit any personal information, browsing data, or user content.

## How We Use Information

This extension operates entirely locally within your browser:

- **CSS Styles**: All CSS styles you input are processed and applied locally in your browser
- **No Data Transmission**: No information is sent to external servers or third parties
- **No Tracking**: We do not track your browsing activity or website visits
- **No Analytics**: No usage analytics or telemetry data is collected

## Data Storage

- **Local Storage Only**: All extension data is stored locally in your browser
- **No Cloud Storage**: We do not use any cloud storage services
- **No Cookies**: The extension does not set or read cookies

## Third-Party Services

This extension does not integrate with or use any third-party services, APIs, or external resources.

## Permissions

The extension only requests the following permission:
- **activeTab**: Required to inject CSS styles into the currently active tab

## Your Rights

- **Complete Control**: You have full control over all CSS styles you inject
- **Easy Removal**: You can remove any injected styles at any time
- **No Data Retention**: We do not retain any of your data

## Children's Privacy

This extension is safe for users of all ages and does not collect any information from children.
